# Habibi City Visual Setup Guide

This guide explains how to get the intended Habibi City visual experience in Minecraft Java Edition 26.2.

## Server-side visuals

The server already provides:

- Faithful 32x server resource pack
- Dynamic Lights datapack
- Vanilla+ city visuals

When you join the server, accept the resource pack prompt.

## Best client-side visual setup

For the full realistic lighting experience, install:

- Fabric Loader 26.2
- Iris + Sodium
- Complementary Unbound r5.8.1

This gives realistic lighting, shadows, bloom, water, and night atmosphere while keeping the game looking like Minecraft.

## Recommended setup levels

### Basic
Accept the server resource pack when joining.

### Recommended
Use Iris + Sodium with Complementary Unbound.

### Lower-performance option
Use Complementary Reimagined instead of Complementary Unbound.

## Important note

Shaders are client-side in Minecraft Java Edition. The server can push a resource pack, but it cannot force real shader lighting or ray tracing.
