# Installing Iris + Sodium for Minecraft 26.2

## What Iris does

Iris lets Minecraft Java Edition use shader packs. Sodium improves performance.

## Why Java may be needed

Minecraft includes its own Java runtime, but the Iris installer may still need Java installed on Windows to open correctly.

If the Iris installer does nothing when clicked, install Java 21 for Windows, then try again.

## Step 1: Download the Iris installer

Use the Windows Iris installer.

Do not download the manual `iris-fabric-...jar` unless you know how to install Fabric mods manually.

## Step 2: Run the installer

Select:

- Game version: 26.2
- Enable Fabric mod support: checked
- Install to: Minecraft Launcher

Click Install.

## Step 3: When it says Completed

Once the Iris installer says Completed, you can close the Iris installer window.

## Step 4: Open Minecraft Launcher

In the launcher, the new profile may show as:

`Fabric Loader 26.2`

That is correct. It may not say “Iris & Sodium.”

Click Play using the Fabric Loader 26.2 profile.

## Step 5: Confirm Iris is installed

In Minecraft, go to:

Options → Video Settings → Shader Packs

If you see the Shader Packs menu, Iris is working.
